# Customer + Integration Logos

These SVG files are clean, monochrome wordmark placeholders sized at a consistent 44px height. They render in any color via `currentColor`, so they inherit the surrounding text color (lets us tint them in the proof bar and brand them on dark backgrounds).

## To swap in official logo files

1. Place the official SVG (or optimized PNG) in this directory using the same filename
2. The HTML references won't change — they all use `assets/logos/{name}.svg`
3. If using PNG, update the `src` attribute extension in HTML
